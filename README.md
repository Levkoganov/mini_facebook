# mini_facebook
this is a MERN Stack project.

# How to install:

## Server:
server is running on: http://localhost:5000
1) CD into the "server" folder
2) npm install.
3) npm start OR npm run dev.

## Client:
1) CD into the "client" folder
2) npm install.
3) npm start.
